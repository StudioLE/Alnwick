# Install the CLI

## Homebrew (Linux and MacOS)

Install the CLI using Homebrew:

```bash
brew install studiole/tap/alnwick
```

Run a command:

```bash
alnwick --help
```

## Docker

Pull the container from GitHub:

```bash
docker pull ghcr.io/studiole/alnwick
```

Refer to [BUILD-CLI-DOCKER.md](BUILD-CLI-DOCKER.md) for recommendations on running the CLI container.
